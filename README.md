# PhraseGenerator
Generate an English adjective-noun pair as a phrase (e.g. SubmissiveMonkey).

## Usage
```php
<?php

require_once __DIR__.'/vendor/autoload.php';

use Badcow\PhraseGenerator\PhraseGenerator;

echo PhraseGenerator::generate();

```
